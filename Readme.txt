# Mi Emprendimiento Personal

**Descripción:**

Esta es la página web de mi emprendimiento, desarrollada con HTML, CSS, Bootstrap y JavaScript. Es el inicio de un proyecto ambicioso que busca convertirse en una plataforma completa.

**Tecnologías Utilizadas:**

* **Frontend:** HTML, CSS, Bootstrap, JavaScript
* **Backend:** (En desarrollo) Angular, MariaDB, Java Spring.

**Objetivos a Futuro:**

* **Implementar Angular:** Para crear una interfaz de usuario más dinámica y escalable.
* **Integrar MariaDB:** Como base de datos para almacenar la información del sitio y la de mis clientes.
* **Utilizar Java Spring:** Para desarrollar el backend y gestionar las solicitudes del servidor.

**Estructura del Proyecto:**

* **public:** Contiene los archivos estáticos (HTML, CSS, imágenes).
* **src:** Contiene el código fuente del proyecto.
* **db:** (En desarrollo) Contiene los scripts para crear y gestionar la base de datos.
* **backend:** (En desarrollo) Contiene el código del backend en Java Spring.
